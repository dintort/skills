# Skills

A collection of useful skills for AI agents:

- **[nanocaveman](skills/nanocaveman/README.md)**: Compressed communication mode. Cuts token usage by omitting redundant fluff.
- **[roast](skills/roast/SKILL.md)**: Runs a merciless AI code review on the current branch against the remote default branch or a custom base branch. Checks for bugs, security vulnerabilities, performance issues, and more. Generates a Markdown report.

## Installation

```bash
npx skills add dintort/skills --global
```
